Stage timing essentials

Basics

How is this different from OnStage Timer Pro?

+

Not Another Stage Timer is the lightweight free version: count up, count down, basic warnings, no Live Activities. OnStage Timer Pro adds Live Activities, Watch sync, audio cues, external display mirror, history, and stage-display features for paid users.

Can I use it for sports timing?

+

Yes. Up-counter mode is ideal for split timing in workouts. Down-counter for interval training. The big-digit display reads from across a gym.

トラブルシューティング

Screen dimmed during my talk

+

Open Settings → Display → Auto-Lock → set to "Never" before going on stage, then restore your preferred setting after. Not Another Stage Timer doesn't override auto-lock by default — that's an OnStage Timer Pro feature.

Why no audio cues?

+

無料 tier shows visual warnings only. For audio cues + override-silent-switch, see OnStage Timer Pro.
